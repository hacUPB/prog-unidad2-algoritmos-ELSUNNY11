### Parte 1: Identificar Algoritmos

Responde si los siguientes enunciados representan un algoritmo. Justifica la respuesta:

1. Una página web. NO es un algoritmo
Porque no es una secuencia de pasos ordenados para resolver un problema; es un medio de información o interacción.  
2. Una receta para hacer un pastel, donde se indican ingredientes y pasos a seguir. SÍ es un algoritmo
Tiene pasos definidos, orden lógico y un resultado esperado.  
3. "Piensa en un número y multiplícalo por otro". NO es un algoritmo completo
Es una instrucción, pero no define claramente entradas, pasos detallados ni final.  
4. Un manual de instrucciones para armar un mueble, con pasos detallados y un orden claro. SÍ es un algoritmo
Presenta instrucciones secuenciales para lograr un objetivo específico.  
5. Una lista de compras organizada en orden alfabético NO es un algoritmo
Es información organizada, pero no describe un proceso paso a paso.  

### Parte 2: Variables y Constantes

1. El valor de la gravedad en la Tierra, 9.8 m/s². Constante

### Parte 5: Estructuras de Control

Describe para qué sirven las estructuras de control. Redacta dos ejemplos, uno de tu vida diaria, es decir cuando tienes que tomar decisiones en tus actividades diarias y oto ejemplo en el que se tengan que utilizar cálculos matemáticos para tomar una u otra decisión.    
¿Para qué sirven?
Permiten controlar el flujo de ejecución de un algoritmo: decidir, repetir acciones o ejecutar pasos según condiciones.

Ejemplo vida diaria (decisión):
Si está lloviendo → llevo paraguas.
Si no → salgo normalmente.

Ejemplo con cálculo matemático:
Si el promedio de notas ≥ 3.0 → apruebo la materia.
Si es menor → debo repetirla.
